Projeto do Site!!!

  • O projeto tem como objetivo principal criar uma plataforma dedicada à venda de discos de vinil, focada em atender colecionadores e amantes da música que buscam adquirir álbuns de forma acessível e prática. Além da venda, o site se propõe a facilitar o acesso a informações detalhadas sobre os produtos, fornecendo descrições dos discos, com dados sobre álbuns e edições. A experiência do usuário é um ponto central, com a meta de desenvolver uma interface intuitiva e agradável, que permita a navegação e realização de compras de maneira eficiente.
  • O público-alvo do projeto é composto por dois grupos principais: colecionadores de vinil, que procuram por edições limitadas, discos raros ou clássicos; e amantes da música em geral, que valorizam a qualidade sonora e o charme do vinil.
  • Para a parte tecnológica, o front-end do site foi desenvolvido utilizando HTML5, garantindo uma estrutura semântica adequada do conteúdo, e CSS3, que permite a estilização e criação de um design responsivo para diferentes dispositivos.
  • O site é acessível a partir de qualquer dispositivo com acesso à internet, seja desktop, tablet ou smartphone. Considerando diferentes condições de conexão, foi priorizado o desenvolvimento de uma plataforma leve, com baixa demanda por gráficos pesados, garantindo a acessibilidade mesmo para usuários com conexões de baixa a média velocidade.
  • Uma das dificuldades enfrentadas no desenvolvimento do site foi a limitação técnica que resultou na necessidade de criar uma página individual para cada produto. Essa limitação também reflete um dos pontos fracos do projeto, que contrasta com os pontos fortes, como o layout responsivo e moderno, e a navegação simplificada.
  • A atualização do catálogo de produtos será realizada de acordo com os novos lançamentos. Além disso, haverá uma atualização trimestral para promoções, e atualizações frequentes de segurança, especialmente para garantir a proteção de dados dos usuários e a segurança nos pagamentos.
  • Vale ressaltar que, por restrições no desenvolvimento, o uso do framework Bootstrap não foi permitido neste projeto, o que desafiou a equipe a criar soluções customizadas para garantir a responsividade e o design do site.
